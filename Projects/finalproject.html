Introduction

The project on this page provides an overview of the final project we did for our class. This project is done by a group of 3 people: Stephanie (Hiu Man) Lam, , Chloe (Jieyi) Chen, and Janet Loyola


With our passion of traving, we are interested in helping the others to discover the most outstanding places in California focusing on only 3 categories:
hotels, restaurants, and landmarks. Our goal of the project is to provide a travel guide for California visitors by showing them the best hotels, restaurants, and landmarks with the applications of web data processing, interactive data vistualization, natural language processing, and interactive user input using Python.
